This repo holds the current metadata for the Columbia University Libraries' Geospatial collections.  These files are in the original XML format and are used for ingest into the Library [Geoblacklight catalog](https://geodata.library.columbia.edu/).  

The actual discovery metadata in the catalog is derived from these files.  The actual catalog metadata is isn JSON and the collection is held in the [OpenGeoMetadata repo](https://github.com/OpenGeoMetadata).  However, at present that archive is out of date.  It should be refreshed by the beginning of the 26-27 academic year.  

This repo is primarily for internal use, if you actually need information here (or otherwise have questions), please just reach our to data@columbia.edu
